Lists in HTML

There are three types of lists in HTML:

  • Unordered List
  • Ordered List
  • Description List

The list above is an unordered list whose order does not matter. This is simply a list of related items.

Below is an example of an Ordered list whose order matters that's why they are displayed using numbers

  1. Head North
  2. Turn Left
  3. Walk Right

You can use Start attribute to specify the number the list should start from.

  1. Item 1
  2. Item 2
  3. Item 3

Reversed attribute

  1. Item1
  2. Item2
  3. Item3
  4. Item4
  5. Item5

Valueattribute is used within a specific list item to change the value of a list item

  1. One
  2. Nine
  3. Ten
List items below will change according to the value of the previous list item

Description list can be used to outline terms and their description as in a glossary.

Study
Studying is an act of studying
The devotion of time and attention to acquiring knowledge on an academic subject, especially by means of books
Work
Business
A person's regular occupation, profession, or trade
